# Insurance Sales Dashboard

Dashboard sprzedażowy przygotowany w Power BI dla fikcyjnej firmy ubezpieczeniowej.  
Projekt pokazuje analizę sprzedaży polis ubezpieczeniowych i powiązanych z tym KPI na poziomie stategicznym przedsiębiorstwa (Executive Summary) oraz menedżerskim / operacyjnym.

> Repozytorium zawiera plik `.pbix`, dane źródłowe, zrzuty ekranu, miary DAX oraz dokumentację projektu.

## Cel projektu

Celem projektu było zbudowanie raportu Power BI przypominającego rozwiązanie analityczne dla obszaru sprzedaży w firmie ubezpieczeniowej.

Raport pozwala analizować m.in:
- wyniki sprzedaży polis,
- wysokość składki: przypisanej i opłaconej
- prowizję agentów,
- strukturę sprzedaży według produktów,
- wyniki sprzedaży na poziomie regionów (w tym oddziałów i agentów)
- zmianę sprzedaży rok do roku,
- udział jednostek sprzedażowych w regionie.

## Zawartość repozytorium

- `report/` — plik Power BI `.pbix`
- `data/` — dane źródłowe w formacie CSV
- `assets/` — zrzuty ekranu raportu
- `docs/` — dokumentacja modelu danych i miar DAX

## Technologie
